# Aivora
AI analytics application
The program we have developed is a unique solution designed for farmers and agricultural professionals. It currently supports seven agents and features an integrated information center that provides the latest updates from the agricultural sector.

Aivora is capable of analyzing plant descriptions and delivering comprehensive insights about them.
Additionally, the application includes advanced analytics powered by artificial intelligence.

The platform is built using C# .NET, and the AI functions as an agent, relying exclusively on the information curated by our team.

The system is deployed on Gemini GPT-5 models (07.08.2025) and hosted on Microsoft Azure.

All seven agents operate within a unified orchestration framework: when a user submits a query, the primary agent (the conductor) determines which specialized agent should generate the response.
